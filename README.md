# lik_wuerdigung #

### Nachteile ###
 Ein grosser Nachteil ist, das der Landesindex im vergleich zu früheren Jahren  
 Relevanz verliert, denn für einige Produkte wie z.B. Cerealien gibt es gar keinen  
 Eintrag in dem LIK von 1959, jedoch im neusten von 2020 gibt es den Eintrag Cerealien  
 mit einer Gewichtung von 0.091%. Des Weiteren können die Vergleiche an Bedeutungskraft  
 verlieren, wenn die Preise von z.B. Importprodukten runter geht, denn Heutzutage  
 wir eine viel Bessere Infrastruktur für den Transport der Güter, welche wir vor  
 61 Jahren noch nicht hatten, so werden Güter günstiger und dann weniger Gewichtet  
 als früher.

## Schlusswort ##

Der Landesindex für Konsumentenpreise ist eine geeignete   Statistik, wenn man  
sie über eine kurze Anzahl von Jahren  miteinander vergleicht, jedoch werden  
Vergleiche über viele Jahre weniger Bedeutungskraft haben, da der Landesindex  
sich über die Jahre verändert somit Vergleiche unschlüssig werden.
